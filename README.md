# MuOS AI AppBuilder

MuOS AI AppBuilder is a project workspace for creating a reusable AI-assisted workflow for building and debugging muOS applications on RG35XX-class handhelds.

## Purpose

This project turns hard-won muOS experience into reusable assets:

- project planning templates
- prompt assets
- debugging playbooks
- packaging templates
- release checklists
- memory scaffolding

## Current Status

The first version includes:

- diagnostics cookbook

- package layout templates

- workflow guide

## Suggested Use

When starting a new muOS app project:

1. read the project plan
2. use the project intake template
3. follow the workflow guide
4. use the diagnostics cookbook during live debugging
5. package using the provided templates
6. maintain project memory throughout the work
